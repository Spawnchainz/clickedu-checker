# clickedu-checker
## Una herramienta de bruteforce para la página escolar "ClickEdu" del Colegio Rosalía.

# Modos de empleo:

- ### Una sola cuenta (verificar):
```bash
python3 main.py <username> <contraseña>
```
- ### Bruteforce:
```bash
./exec.sh <username> <nombre_del_archivo_de_la_wordlist>
```
